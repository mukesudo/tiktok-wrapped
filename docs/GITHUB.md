# GitHub Polish

This file is the practical checklist for making the repository look strong on GitHub.

## Recommended Repository Description

Use this as the repo description:

> Spotify Wrapped-inspired TikTok analytics app built with Next.js, TypeScript, Tailwind, and local-first JSON parsing.

## Recommended Website Field

Use your deployed production URL, for example:

```text
https://your-domain.com
```

## Recommended Topics

Add these topics in the GitHub repo settings:

- `nextjs`
- `react`
- `typescript`
- `tailwindcss`
- `framer-motion`
- `tiktok`
- `analytics`
- `data-visualization`
- `wrapped`
- `app-router`
- `vercel`
- `json`

## Recommended Social Preview

GitHub social preview is configured in the repository settings, not from code.

Use one of these:

1. Export a PNG from `docs/assets/repo-banner.svg`
2. Use a screenshot of the deployed home page
3. Capture the deployed `/opengraph-image` output once the site is live

## Recommended Pin Line

If you pin this repo on your profile, use:

> TikTok Wrapped analytics app with replay insights, search themes, and a share-ready full-screen recap.
